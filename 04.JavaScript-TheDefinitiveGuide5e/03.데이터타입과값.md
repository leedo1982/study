3장 데이터 타입과 값  
자바스크립트에서는 세가지 기본 데이터 타입인 숫자, 텍스트으 ㅣ나열, 불리언 진리값 으로 원하는 작업을 수행한다.  
또한 두개의 단순 테이터 타입인 null과 undefined을 정의한다.  
기본적인 데이터 타입 이외에도 객체라 불리는 복합 데이터 타입을 지원한다.  
객체는 값들의 모임을 표현한다.  
자바스크립트에서 객체는 이원적인 성질이 있다.  
객체는 이름 붙은 값들의 순서없는 모임을 나타내거나, 번호가 붙은 순서 있는 값들의 모임을 나타내기도 한다.  
후자의 경우 이러한 객체를 배열이라고 부른다.  
객체와 배열은 근본적으로 동일한 데이터 타입이지만, 개별적으로 취급할 것이다.  
자바스크립트는 함수라고 불리는 또다른 특별한 형태의 객체를 정의한다.  
함수는 실행 가능한 코드를 담은 객체이다.

3.1 숫자  
자바 스크립트는 정수값과 실수값을 구별하지 않는다는 점에서 C나 java와 다르다.
자바 스크립트에서 모든 숫자는 실수 로 표현된다.
숫자는 IEEE 754표준에 의해 정의되는 64비트 실수 형식을 사용하여 표현한다.
따라서 +-1.7976931348623157 x 10 308 만틈 크고 +-5x10 -324 만큼 작은 숫자를 표현할수 있다.

3.1.1 정수 리터럴
자바스크립트에서 10진수 정수는 숫자 시퀀스형태로 작성된다.
정확히 -2e53 ~ 2e53 까지 표현가능
몇몇 정수 연산자들은 -2e31 ~ (2e31-1)에 이르는 32비트 정수에 대해서만 작동함을 유의

3.1.2 16진수와 8진수
16진수 : '0x' 또는  '0X' 시작
8진수 : 0으로 시작하고 0부터 7사이 숫자 시퀀스가 뒤따르는 형태이다.
따라서 0으로 시작하는 정수 리터럴은 절대로 사용하지 말라. 10진수로 인식할지 8진수로 인식할지 알수 없다.

3.1.3 부동소수점 리터럴
부동소수점 리터럴은 전통적인 문법으로 표현됨 실수는 정수부분과, 소수점, 소수점이하 부분으로 표현

3.1.5 숫자 변환
자바스크립트는 숫자를 문자열 형식으로 변환하거나 문자열을 숫자로 변환할 수 있다.

3.1.6 특수한 숫자 값
수리연산이 정의되지 않은 결과를 산출하거나 에러를 발생시킬 경우에 특수한 자바스크립트 값이 NaN으로 출력한다.
어떤 숫자와 비교해도 동일하지 않으며
심지어 스스로와 비교해도 그렇다.
이값을 테스트해보기 위해 isNaN()이라는 특별한 함수가 필요하다.
관련함수 isFinite()는 주어진 숫자가 NaN이 아닌 동시에 양의 무한대나 음의 무한대가 아닌지 여부를 검사한다.

특수한 숫자상수들
Infinity : 무한대를 나타내는 특수한 값
NaN  : 숫자가 아닌 특수한값
Number.MAX_VALUE : 표현 가능한 가장큰수
Number.MIN_VALUE : 표현 가능한(0에 가까운)  가장 작은수
Number.NaN : 숫자가 아닌 특수한 값
Number.POSITIVE_INFINITY : 양의 무한대를 표현하는 특수한 값
Number.NEGATIVE_INFINITY : 음의 무한대를 표현하는 특수한 값
