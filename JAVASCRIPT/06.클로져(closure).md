1.정의  
외부함수(포하마고 있는)의 변수에 접근할 수 있는 내부 함수.  
클로저는 독립적인(자유)변수를 가리키는 함수 이다. 또는 클로저 안에서 정의된 함수는 만들어진 환경을 기억한다.  

2.어휘의 범위  
```
function init(){
  var name = "모질라";
  function displayName(){ // 내부함수, 즉 클로져인 displayName()
    alert(name);
  }
  displayName();
}
init();
```
